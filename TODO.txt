# Детальна інструкція: AUR_GUIDE.md

# Коротко:
1. Зареєструйся на https://aur.archlinux.org/
2. Додай SSH ключ в акаунт
3. Клонуй AUR репо:
   git clone ssh://aur@aur.archlinux.org/mod-manager-git.git
   
4. Скопіюй PKGBUILD (вже готовий!):
   cp PKGBUILD mod-manager-git/
   cd mod-manager-git
   
5. Відредагуй PKGBUILD (поміняй yourusername на свій)
   
6. Запуш:
   makepkg --printsrcinfo > .SRCINFO
   git add PKGBUILD .SRCINFO
   git commit -m "Initial import"
   git push

✅ Готово! Тепер всі Arch користувачі можуть:
   yay -S mod-manager-git
