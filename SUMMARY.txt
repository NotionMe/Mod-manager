1️⃣  PUSH НА GITHUB
   ───────────────
   git remote add origin https://github.com/yourusername/mod-manager.git
   git add .
   git commit -m "Ready for distribution"
   git push -u origin main
   
   # Створи release
   git tag -a v1.0.0 -m "First public release"
   git push origin v1.0.0


2️⃣  ОПУБЛІКУЙ В AUR (опціонально, але круто)
   ──────────────────────────────────────────
   # Детальна інструкція в AUR_GUIDE.md
   
   Коротко:
   - Зареєструйся на aur.archlinux.org
   - Додай SSH ключ
   - git clone ssh://aur@aur.archlinux.org/mod-manager-git.git
   - Скопіюй PKGBUILD туди
   - Відредагуй (поміняй yourusername)
   - makepkg --printsrcinfo > .SRCINFO
   - git add . && git commit -m "Initial" && git push
